# 🖥️ T01: Aula de Formació per ONG — “Horitzons Digitals”
**Autor:** Pol Serrano, Pau Constanseu i Blai Vergés
**Data:** 11/05/2026  

En aquesta activitat s’ha desenvolupat una proposta completa de **transformació digital** per a l’ONG Horitzons Digitals, amb l’objectiu de crear una aula de formació tecnològica destinada a col·lectius vulnerables 💻🌍.

El projecte parteix d’una situació inicial on l’organització disposava únicament d’un local buit, sense infraestructura tecnològica ni equipament informàtic. A partir d’aquesta necessitat, s’ha realitzat una anàlisi global per definir una solució funcional, segura i econòmicament viable 📊.

Durant el desenvolupament del projecte s’han estudiat les necessitats dels diferents perfils d’usuaris, com persones grans, persones en situació d’atur i joves en formació tecnològica 👥. Això ha permès adaptar tant el maquinari com el programari a les característiques i necessitats reals de cada col·lectiu.

També s’ha dissenyat tota la infraestructura de xarxa de l’aula, incloent ordinadors, dispositius de xarxa, servidor NAS, connectivitat WiFi i segmentació mitjançant VLANs 🌐. A més, s’han incorporat mesures de seguretat i estratègies de còpies de seguretat per garantir la protecció de les dades i la continuïtat del servei 🔒.

Una part important del projecte ha estat la comparativa entre programari lliure i programari comercial, valorant aspectes com el cost, la seguretat, el manteniment i la facilitat d’ús ⚖️. Finalment, s’ha recomanat una solució basada en Linux Mint i LibreOffice per reduir costos i facilitar el manteniment de l’aula.

Finalment, s’ha elaborat un pressupost detallat i un estudi de viabilitat tècnica i econòmica, demostrant que el projecte és sostenible i assumible dins les limitacions pressupostàries de l’ONG 💰🚀.

Aquesta activitat reflecteix la importància de planificar correctament una infraestructura tecnològica real, tenint en compte no només els aspectes tècnics, sinó també els factors socials, econòmics i de sostenibilitat 📚.
